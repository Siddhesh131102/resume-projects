# Power BI Data Analysis Project

##  Project Overview
This Power BI project analyzes business performance using multiple connected datasets. The goal is to uncover insights related to revenue, customers, products, returns, and store operations through interactive dashboards and KPIs.

##  Dataset Description
The project consists of the following tables:

- **Calendar**
- **Customer**
- **Product**
- **Regions**
- **Return Data**
- **Store**
- **Transaction Data**

### Data Transformation & Cleaning
- **Corrected data types**
- **Trimmed and cleaned text fields**
- **Created relationships in Model View**
- **Created a dedicated Measure Table for DAX calculations**

##  Key Measures Created
Some important measures include:

- **Total Revenue**
- **Total Profit**
- **Total Transactions**
- **Total Unique Customers**
- **Previous Month Revenue**
- **Return Rate**
- **Profit Margin**
- **Average Revenue per Customer**
