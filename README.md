# Data Projects-Siddhesh Narkar
This repository contains 3 data analysis projects built using Excel,SQL,and Power BI.

##Project Structure
**Excel** -Business Insights analysis using Pivot Tables,Pivot Charts,Slicers and Advance Formulas.
**SQL** -SQL queries including months by revenue,movies above average rental rate,movies not available in inventory,movies rented per store,top 10 customes by payment and top 10 movies by revenue
**Power BI** -Fully interactive dashboards showing business insights and KPIs.

##Project Summary
###Excel Project
This Excel project provides business insights using Pivot Tables,Pivot Charts and Advance Formulas.It includs KPIs like total sales,total quantity,revenue by product line,top performing countries and year filters.

###SQL Project
Contains SQL queries such as:
-Film rental business queries
-Inner Joins between multiple tables
-Group by and Order By

###Power BI Project
Business Insights dashboard built using Power BI Desktop.
This report includes 5 dashboards:
1.Business Sumaary
2.Customer Analysis
3.Product Performance
4.Store Performance
5.Geographical Analysis

##How to Run
**Excel** -Open the Excel file in Microsoft Excel
**SQL** -Run in any SQL tool.
**Power BI** -Open in Power BI Desktop
